Vous souhaitez donner 45 min de votre temps afin que nos enfants puissent passer une kermesse inoubliable !
Voici les stands de jeux et créneaux ci-dessous.
Rien de plus simple, cochez la (les) cases du stand puis inscrivez votre nom/prénom et mail dans le message suivant.
Pas d'inquiétude, les règles des jeux sont simples.
